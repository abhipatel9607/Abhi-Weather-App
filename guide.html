It is a meteorological application which provides all weather related information to the user.
Search box has been given in the top right corner of this web application where the user can go
and get the weather information of his favorite place.
There is also a Dropdown component in the Navbar section of the website, where by clicking user
can get all the weather information of the given cities, in addition,
weather information of some of the main cities of India has been given at the bottom of the website.
